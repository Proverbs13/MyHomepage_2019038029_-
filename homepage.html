보유 기술




  • Python3

  • 알고리즘 문제 해결 방식으로 Python 3를 사용해 여러가지 문제를 해결했습니다.
  • Python 외부라이브러리인 Pygame으로 공포게임 프로젝트를 진행했습니다.


  • Android

  • 안드로이드 앱 개발 언어 자바 문법에 이해도가 있습니다.
  • 자바 기반 안드로이드 프론트엔드 개발능력이 있습니다.
  • 장애인을 위한 편의시설 제공 앱개발 프로젝트에 프론트엔드로 참여한 경험이 있습니다.


  • Spring Boot

  • 웹 백엔드 개발을 위한 Spring boot 인프런강의를 수강하며 공부하고 있습니다.
